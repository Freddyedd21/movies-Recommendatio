📖 Acerca del Sistema

MovieRecs es un sistema de recomendación de películas basado en clusters generados con Apache Spark.

🎯 Características:

  • Segmentación de usuarios con K-Means
  • Recomendaciones basadas en clusters
  • Métricas de evaluación: Precision@K y Recall@K
  • Dataset: MovieLens 1M (1,000,000 calificaciones)

🏷️ Clusters Identificados:

  • Cluster 0-1: Usuarios ocasionales/intermedios
  • Cluster 2-4: Usuarios activos
  • Cluster 5-7: Usuarios muy activos

📚 Documentación API

Endpoints disponibles:

GET /health

Verificar estado de la API

Ejemplo respuesta 
{"status": "ok", "users_loaded": 5}
GET /recommendations/{user_id}

Obtener recomendaciones de un usuario específico

Ejemplo respuesta 
{"user_id": 1, "cluster": 1, "recommendations": [...]}
